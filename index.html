    1.1- Introducción

    Normalmente existe cierta confusión entre los términos "Página web (website)" y "Blog", trataré de aclarar de manera concisa algunas de las diferencias existentes.

    * Página web:

    Una página web es estática: su contenido no se actualiza con frecuencia. Y básicamente sirven para vender o dar a conocer una marca, una empresa, o productos concretos.

    • Mayormente requiere de un buen nivel de programación.
    • Es más versátil que un blog ... por ejemplo, podemos colocar un reloj digital, acceso a pagos con tarjeta, insertar hipervínculos, etc. El límite es la imaginación.
    • Tendencia a ser unidireccional (No existe diálogo entre el autor y los receptores), lo cual es parcialmente malo.

    * Blog:

    En un blog, el autor escribe artículos sobre temas que le interesan como películas, negocios, música, moda, deportes, juegos, marketing, psicología, cocina, etc. Y lo más importante de todo: el contenido se actualiza con bastante frecuencia.

    • Es de fácil manejo, no requiere de conocimientos sobre programación.
    • Presenta limitaciones, dependerá de la página web que se utilize para la creación de esta ... entre ellas tenemos wix, blogger, google forms, etc.
    • Normalmente existe una continua interacción entre el autor y sus seguidores (contenido dinámico).
